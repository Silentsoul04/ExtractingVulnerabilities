Delta Electronics TPEditor Versions 1.97 and prior. An improper input
validation may be exploited by processing a specially crafted project
file not validated when the data is entered by a user. Successful
exploitation of this vulnerability may allow an attacker to
read/modify information, execute arbitrary code, and/or crash the
application.
